# Compliance-to-Architecture / ontology

JSON-LD ontology + schemas for the Compliance-to-Architecture Framework™. 12-layer ontology: Authority → Obligation → Control → Mitigation → Architecture → Policy-as-Code → Evidence → Audit-Trail.
